== Description ==

bizzCam video Allows bizzCam customers to easily embed the bizzCam code, and send short bizzClip video commercials to their website.



== Installation ==

Upload the bizzCam video plugin to your blog, Activate it.

After activation a new menu option will added with the name "bizzCam", below menu option named "Settings".
To enter the embed code click on the menu option "bizzCam" and enter the code in the form.
To show th latest video on the website just add the short code "[front_view]" on the required page.
